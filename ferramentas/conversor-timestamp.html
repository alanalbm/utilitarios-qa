Conversor de Timestamp

Converta Unix timestamp, ISO 8601 e datas locais rapidamente.

Informe um valor ou use a data atual.

Segundos ou milissegundos?

Muitos sistemas Unix usam segundos desde 1º de janeiro de 1970. JavaScript normalmente trabalha com milissegundos. A diferença de três zeros é uma causa comum de datas incorretas.

Uso responsável: Ao testar integrações, confirme também o fuso horário e se o contrato exige UTC ou horário local.

Privacidade por padrão

Os valores desta ferramenta são processados pelo JavaScript do navegador. A página não precisa enviar o conteúdo digitado para um servidor para executar a função.
