Gerador de dados fictícios

Gere nomes, e-mails reservados, telefones e IDs em JSON ou CSV.

Os dados fictícios aparecerão aqui.

Massa de teste sintética

Os registros usam nomes combinados aleatoriamente e o domínio reservado example.test. Assim, reduzem o risco de enviar mensagens a pessoas reais durante testes.

Uso responsável: Revise regras específicas do sistema, como tamanho máximo, unicidade e formatos obrigatórios, antes de importar os dados.

Privacidade por padrão

Os valores desta ferramenta são processados pelo JavaScript do navegador. A página não precisa enviar o conteúdo digitado para um servidor para executar a função.

Aplicações em testes de software

  • Preparação de dados e cenários de homologação.
  • Validação manual de contratos de API e interfaces.
  • Criação de exemplos reproduzíveis para documentação.
  • Apoio a automações sem depender de serviços externos.
