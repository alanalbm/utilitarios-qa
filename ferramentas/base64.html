Codificador Base64

Codifique e decodifique texto UTF-8 em Base64 com privacidade.

Base64 não é criptografia

Base64 transforma bytes em texto usando um conjunto limitado de caracteres. É útil para transporte de dados, mas qualquer pessoa pode decodificar o conteúdo.

Uso responsável: Nunca trate Base64 como forma de proteger senhas, tokens ou informações confidenciais.

Privacidade por padrão

Os valores desta ferramenta são processados pelo JavaScript do navegador. A página não precisa enviar o conteúdo digitado para um servidor para executar a função.
