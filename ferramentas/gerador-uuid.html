Gerador de UUID

Crie UUIDs v4 aleatórios para testes, bancos de dados e integrações.

Clique em “Gerar UUIDs”.

O que é UUID v4?

UUID v4 é um identificador de 128 bits gerado com aleatoriedade. Ele é muito usado como chave de registros, identificador de correlação e valor único em sistemas distribuídos.

Uso responsável: A chance de colisão é extremamente pequena, mas um UUID não substitui regras de autorização ou proteção de dados.

Privacidade por padrão

Os valores desta ferramenta são processados pelo JavaScript do navegador. A página não precisa enviar o conteúdo digitado para um servidor para executar a função.

Aplicações em testes de software

  • Preparação de dados e cenários de homologação.
  • Validação manual de contratos de API e interfaces.
  • Criação de exemplos reproduzíveis para documentação.
  • Apoio a automações sem depender de serviços externos.
