Gerador e validador de CPF/CNPJ

Gere e valide CPF e CNPJ matematicamente válidos para ambientes de teste, sem enviar dados a servidores.

000.000.000-00

Como funciona a validação?

CPF e CNPJ possuem dígitos verificadores calculados a partir dos números anteriores. A ferramenta reproduz esse cálculo para criar valores matematicamente válidos. Isso não significa que o documento exista ou pertença a alguém.

Uso responsável: Use apenas em ambientes de desenvolvimento, homologação, demonstração e estudos. Não use números gerados para criar contas reais, enganar serviços ou se passar por outra pessoa.

Privacidade por padrão

Os valores desta ferramenta são processados pelo JavaScript do navegador. A página não precisa enviar o conteúdo digitado para um servidor para executar a função.

Aplicações em testes de software

  • Preparação de dados e cenários de homologação.
  • Validação manual de contratos de API e interfaces.
  • Criação de exemplos reproduzíveis para documentação.
  • Apoio a automações sem depender de serviços externos.
