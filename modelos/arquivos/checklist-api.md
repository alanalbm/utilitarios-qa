# Checklist de Testes de API

- [ ] Contrato, método, URL e headers
- [ ] Status e corpo esperado
- [ ] Campos obrigatórios, tipos, nulos e limites
- [ ] Autenticação ausente, inválida, expirada
- [ ] Autorização por perfil e propriedade
- [ ] Recurso inexistente e identificador malformado
- [ ] Duplicidade, idempotência e concorrência
- [ ] Paginação, filtro e ordenação
- [ ] Tempo, limites e comportamento sob falha
- [ ] Logs e respostas sem segredos
