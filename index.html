Uso responsável e privacidade

Os dados são enviados para algum servidor?

Não. As ferramentas principais funcionam localmente por JavaScript no navegador. O projeto não precisa receber os valores digitados.

Posso usar os dados gerados em produção?

Não. Os geradores existem para testes, estudos e desenvolvimento. Utilize dados reais apenas com base legal, autorização e controles adequados.

As ferramentas são gratuitas?

Sim. O acesso é gratuito. Futuramente, anúncios podem ajudar a manter o projeto sem cobrar pelo uso.
