Canal de contato

Use o perfil do projeto no GitHub ou o canal profissional informado no repositório. Ao relatar um problema, inclua a URL da ferramenta, navegador, dispositivo e passos para reproduzir, sem anexar dados pessoais ou arquivos confidenciais.

Importante: o site não oferece suporte a sistemas de terceiros nem recebe documentos para processamento manual.
