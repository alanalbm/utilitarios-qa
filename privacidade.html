Política de privacidade

Última atualização: julho de 2026.

Processamento das ferramentas

Textos e arquivos usados nas funções principais são processados no navegador. O projeto não possui uma API própria para receber esses conteúdos. Algumas bibliotecas JavaScript são distribuídas junto com o site.

Google Analytics

O Analytics é ativado apenas após consentimento. Ele pode registrar páginas acessadas, dispositivo, origem aproximada da visita e eventos de uso. O site foi configurado para não enviar o conteúdo digitado, nomes de arquivos, CPF, CNPJ, JSON, tokens ou resultados das ferramentas como parâmetros de evento.

Armazenamento local

Tema, favoritos, histórico e escolha de métricas podem ser salvos no navegador. Esses dados podem ser apagados pelas configurações do próprio navegador.

Publicidade

Espaços estão preparados, mas anúncios só devem ser ativados após aprovação e atualização desta política com as informações exigidas pelo provedor.

Cuidados do usuário

Evite inserir dados pessoais reais, segredos, documentos confidenciais ou arquivos de produção. Embora o processamento seja local, extensões, dispositivo e navegador também fazem parte do seu ambiente de segurança.

Contato

Use a página de contato para dúvidas relacionadas a esta política.
