Política de privacidade

Como o site trata conteúdo, métricas e armazenamento local.

Última atualização: 17 de julho de 2026.

Processamento local

As ferramentas principais processam os dados no navegador. Não existe API própria recebendo o conteúdo digitado.

Google Analytics

O site usa Google Analytics 4, ID G-MM48KQ55RP. O armazenamento analítico permanece negado até o visitante aceitar as métricas. O conteúdo digitado nas ferramentas não é enviado como evento pelo código do site.

Armazenamento local

Tema, consentimento, favoritos e histórico recente podem ser guardados no dispositivo.

Terceiros

O site é hospedado no GitHub Pages. Anúncios poderão ser ativados no futuro, após revisão desta política.

Controles

É possível recusar métricas, reabrir preferências no rodapé e limpar os dados do navegador.
