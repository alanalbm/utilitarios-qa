How to choose the right tool

Start with the result you need, use fictional data whenever possible and review output before applying it to a real system. These related pages are organized so you can move from preparation to execution and validation without switching between many platforms.

Privacy and performance

The portal prioritizes in-browser processing. Tools that handle large files may use significant memory, so test with a copy and keep the original file safe.
