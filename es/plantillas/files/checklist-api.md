# Checklist de Pruebas de API

- [ ] Contrato, método, URL y cabeceras
- [ ] Estado y cuerpo esperado
- [ ] Campos obligatorios, tipos, nulos y límites
- [ ] Autenticación ausente, inválida y expirada
- [ ] Autorización por rol y propiedad
- [ ] Recurso inexistente e identificador inválido
- [ ] Duplicados, idempotencia y concurrencia
- [ ] Paginación, filtros y ordenación
